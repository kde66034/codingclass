CSS 소스

푸터 메뉴를 display: flex로 나란히 정렬한다. CSS 공통 요소, 폰트 정보 등은 따로 빼 놓았기 때문에 자세히 보고 싶다면 하단의 CSS 소스 보기 버튼을 누를 것. 

<style>
    .footer__menu {
        display: flex;
    }
    .footer__menu > div {
        width: 16%;
    }
    .footer__menu > div h3 {
        font-size: 18px;
        margin-bottom: 20px;
    }
    .footer__menu li a {
        font-size: 14px;
        color: #666;
        margin-bottom: 10px;
        display: block;
    }
    .footer__right {
        border-top: 1px solid #d9d9d9;
        text-align: center;
        padding-top: 40px;
        color: #666;
        line-height: 1.5;
    }
</style>
