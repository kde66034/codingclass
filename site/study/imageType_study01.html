사이트 만들기 : 이미지 유형 01

이번 포스팅에서는 Figma에서 만든 이미지 유형 섹션을 HTML과 CSS 코드로 옮기는 작업을 해보도록 하겠습니다.

Figma

먼저 만들고자 하는 웹 사이트의 모양을 Figma라는 프로그램을 이용하여 아래와 같이 만들어주도록 하자.
이번에 만들 웹 사이트의 유형은 이미지 위에 제목과 본문(설명), 버튼을 올려놓는 간단한 방식이다.

HTML 소스

웹 표준 준수를 위해 시맨틱 태그를 적극 사용하고, 이미지는 img 태그가 아닌 article 태그에 CSS로 나중에 추가해줄 것.
자세한 설명은 아래 예시 코드의 주석처리 내용을 참조할 것. 

<body>
    <section id="imageType01" class="image__wrap nexon section">
        <h1>별빛 제빵교실 개강소식🍞</h1>
        <p>별빛 제빵교실이 2022년 8월 23일 개강합니다. 수강신청 기회는 지금 뿐이니 서두르세요!</p>
        <div class="image__inner container">
            <article class="image img1">
                <h3 class="image__title">나만의 제빵 포트폴리오</h3>
                <p class="image__desc">
                    자신만의 제빵 포트폴리오를 만들 수 있도록 돕겠습니다. 
                    열심히만 한다면 완전 수료할 때 즈음에는 당신도 훌륭한 제빵사!
                </p>
                <a class="image__btn" href="/" title="자세히 보기">
                    자세히 보기
                </a>
            </article>
            <article class="image img2">
                <h3 class="image__title">나만의 제빵 포트폴리오</h3>
                <p class="image__desc">
                    자신만의 제빵 포트폴리오를 만들 수 있도록 돕겠습니다. 
                    열심히만 한다면 완전 수료할 때 즈음에는 당신도 훌륭한 제빵사!
                </p>
                <a class="image__btn" href="/" title="자세히 보기">
                    자세히 보기
                </a>
            </article>
        </div>
    </section>
</body>

CSS 소스

reset에서 margin값과 padding값을 0으로 초기화 시키는 작업부터 시작한다.
또한, HTML에서 추가하지 않았던 이미지를 background: url() 형식으로 추가한다.
자세한 설명은 아래 예시 코드의 주석처리된 내용을 참조할 것. 

<style>
    /* fonts */
    @import url('https://webfontworld.github.io/NexonLv1Gothic/NexonLv1Gothic.css');

    .nexon {
        font-family: 'NexonLv1Gothic';
        font-weight: 400;
    }

    /* reset */
    * {
        margin: 0;
        padding: 0;
    }
    h1, h2, h3, h4, h5, h6 {
        font-weight: normal;
    }
    a {
        text-decoration: none;
        color: #000;
    }
    img {
        width: 100%;    /* 이미지 튀어나온 거 집어넣기 */
    }

    /* common */
    .container {
        width: 1160px;
        padding: 0 20px;
        margin: 0 auto;
        min-width: 1160px;  /* 반응형 할 때 없애줄 것 */
    }
    .section {
        padding: 120px 0;
    }
    .section > h1 {
        font-size: 50px;
        line-height: 1;
        text-align: center;
        margin-bottom: 20px;
    }
    .section > p {
        font-size: 22px;
        font-weight: 300;
        text-align: center;
        margin-bottom: 70px;
        color: #666;
    }

    /* imageType */
    .image__inner {
        display: flex;
        justify-content: space-between;     /* 간격 떨어뜨리기 */
    }
    .image {
        width: 49%;
        height: 370px;
        padding: 200px 30px 30px 30px;
        color: #fff;
        box-sizing: border-box;
    }
    .image.img1 {
        background: url(img/imageType01_bg01.jpg) no-repeat center;
    }
    .image.img2 {
        background: url(img/imageType01_bg02.jpg) no-repeat center;
    }
    .image__title {
        font-size: 32px;
        margin-bottom: 10px;
    }
    .image__desc {
        font-size: 16px;
        line-height: 1.5;
        margin-bottom: 10px;
        font-weight: 300;
        /* 2줄효과 : 글이 넘쳐도 2줄을 넘지 않도록 해줌 */
        overflow: hidden;
        text-overflow: ellipsis;
        display: -webkit-box;
        -webkit-line-clamp: 2;
        -webkit-box-orient: vertical;
        padding-right: 30px;
    }
    .image__btn {
        background-color: rgba(96, 52, 0, 0.8);
        padding: 10px 15px;
        font-size: 16px;
        font-weight: 300;
        color: #fff;
        display: inline-block;
    }
</style>
