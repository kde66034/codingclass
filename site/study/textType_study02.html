사이트 만들기 : 텍스트 유형 STUDY 02

이번 포스팅에서는 Figma에서 만든 텍스트 유형 섹션을 HTML과 CSS 코드로 옮기는 작업을 해보도록 하겠습니다.
아이콘 출력은 아이콘이 들어갈 원을 먼저 코딩으로 만들어 낸 후, 아이콘을 이미지 스프라이트 기법으로 원 안에 넣는 식으로 코딩을 진행하였습니다.

Figma

먼저 만들고자 하는 웹 사이트의 모양을 Figma라는 프로그램을 이용하여 아래와 같이 만들어주도록 하자. 그와 동시에 이미지 스프라이트용 아이콘 SVG를 출력할 페이지도 따로 만들도록 한다. 만드는 방법은 지난 이미지 스프라이트 포스팅을 참조할 것.

HTML 소스

웹 표준 준수를 위해 시맨틱 태그를 적극 사용하도록 하자. img 태그의 alt와 같은 효과를 주기 위해 아이콘이 들어갈 span 태그의 클래스 값을 "ir"로 설정하자. 자세한 설명은 아래 예시 코드의 주석처리 내용을 참조할 것. 

<body>

CSS 소스

reset에서 margin값과 padding값을 0으로 초기화 시키는 작업부터 시작한다. 아이콘 출력은 이미지 스프라이트와 IR 효과를 적극 활용하도록 한다. 자세한 설명은 아래 예시 코드의 주석처리된 내용을 참조할 것. 

<style>
    /* fonts */
    @import url('https://webfontworld.github.io/gmarket/GmarketSans.css');

    .gmarket {
        font-family: 'GmarketSans';
        font-weight: 400;
    }

    /* reset */
    * {
        margin: 0;
        padding: 0;
    }
    h1, h2, h3, h4, h5, h6 {
        font-weight: normal;
    }
    a {
        text-decoration: none;
        color: #000;
    }
    img {
        width: 100%;    /* 이미지 튀어나온 거 집어넣기 */
    }

    /* common */
    .container {
        width: 1600px;
        margin: 0 auto;
        min-width: 1600px;  /* 반응형 할 때 없애줄 것 */
    }
    .section {
        padding: 120px 0;
        display: flex;
        
    }

    .card__title > h2 {
        font-size: 70px;
        line-height: 1.1;
        margin-bottom: 30px;
        font-weight: bold;
        width: 340px;
    }
    .card__title .subtitle {
        border-radius: 16px;
        background-color: #ff3344;
        margin-bottom: 20px;
        width: 202px;
        height: 32px;
    }
    .card__title .subtitle > p {
        font-size: 18px;
        font-weight: 500;
        text-align: center;
        color: #fff;
        padding: 6px 0;
    }
    .main__desc > p {
        font-size: 24px;
        font-weight: 500;
        line-height: 1.4;
        width: 372px;
    }

    /* cardType */
    #card__icon {
        width: 80px;
        height: 80px;
        display: block;
        border-radius: 100%;
        margin-bottom: 10px;
    }
    /* 아이콘 하나하나에 색깔 넣어주기 */
    .icon__logo1 {
        background-color: #dafbeb;
    }
    .icon__logo2 {
        background-color: #e8ffd6;
    }
    .icon__logo3 {
        background-color: #fde2e2;
    }

    }

    }

    /* 이미지 스프라이트 */
