사이트 만들기 : 텍스트 유형 STUDY 03

이번 포스팅에서는 Figma에서 만든 텍스트 유형 섹션을 HTML과 CSS 코드로 옮기는 작업을 해보도록 하겠습니다.
아이콘 출력은 아이콘이 들어갈 원을 먼저 코딩으로 만들어 낸 후, 아이콘을 이미지 스프라이트 기법으로 원 안에 넣는 식으로 코딩을 진행하였습니다.

Figma

먼저 만들고자 하는 웹 사이트의 모양을 Figma라는 프로그램을 이용하여 아래와 같이 만들어주도록 하자. 그와 동시에 이미지 스프라이트용 아이콘 SVG를 출력할 페이지도 따로 만들도록 한다. 만드는 방법은 지난 이미지 스프라이트 포스팅을 참조할 것.

CSS 소스

reset에서 margin값과 padding값을 0으로 초기화 시키는 작업부터 시작한다. 아이콘 출력은 이미지 스프라이트와 IR 효과를 적극 활용하도록 한다. 자세한 설명은 아래 예시 코드의 주석처리된 내용을 참조할 것. 

<style>
    /* fonts */
    @import url('https://webfontworld.github.io/gmarket/GmarketSans.css');

    .gmarket {
        font-family: 'GmarketSans';
        font-weight: 400;
    }

    /* reset */
    * {
        margin: 0;
        padding: 0;
    }
    body {
        background-color: #FDF6F6;
    }
    h1, h2, h3, h4, h5, h6 {
        font-weight: normal;
    }
    a {
        text-decoration: none;
        color: #fff;
    }
    img {
        width: 100%;    /* 이미지 튀어나온 거 집어넣기 */
    }
    hr {
        margin: 30px 0;
    }
    button {
        border: none;
        font-family: 'GmarketSans';
        cursor: pointer;
    }

    /* common */
    .container {
        width: 1160px;
        padding: 0 20px;
        margin: 20px auto;
        min-width: 1160px;  /* 반응형 할 때 없애줄 것 */
    }
    .section {
        padding: 120px 0;
    }
    .section > h2 {
        font-size: 50px;
        line-height: 1;
        text-align: center;
        margin-bottom: 30px;
    }
    .section > p {
        font-size: 24px;
        text-align: center;
        margin-bottom: 70px;
        font-weight: 500;
        color: #666;
    }

    /* cardType */
    .card__inner {
        display: flex;
        justify-content: space-between;     /* 간격 떨어뜨리기 */
    }
    .card {
        width: 33%;
        background: #fff;
        border-radius: 10px;
        margin-right: 20px;
    }
    #card__icon {
        width: 60px;
        height: 60px;
        display: block;
        border-radius: 100%;
        margin-right: 10px;
    }
    /* 아이콘 하나하나에 색깔 넣어주기 */
    .icon__logo1 {
        background-color: #fceceb;
    }
    .icon__logo2 {
        background-color: #ecfceb;
    }
    .icon__logo3 {
        background-color: #fcebfb;
    }

    .card__body {
        padding: 30px;
    }
    .card__body .title {
        font-size: 24px;
        margin-bottom: 10px;
        font-weight: 500;
        /* 한 줄 효과 */
        overflow: hidden;
        white-space: nowrap;
        text-overflow: ellipsis;
        padding-right: 30px;    /* 제목 길어져도 한 줄 유지 */
    }
    .card__body .desc {
        font-size: 20px;
        font-weight: 500;
        line-height: 1.4;
        /* 7줄효과 : 글이 넘쳐도 7줄을 넘지 않도록 해줌 */
        overflow: hidden;
        text-overflow: ellipsis;
        display: -webkit-box;
        -webkit-line-clamp: 7;  /* 7줄까지 제한 */
        -webkit-box-orient: vertical;
    }
    .card__body .more {
        margin: 0 82px;
        font-size: 18px;
        font-weight: 500;
        color: #fff;
        border-radius: 17px;
        background-color: #e1a39f;

    }

    .profile {
        display: flex;
    }
    .user__name > h2 {
        font-size: 28px;
        font-weight: 500;
        position: relative;
    }

        color: #666;

    /* ir 효과 (이미지 대체 효과 == alt) */
    /* 화면상에서는 안나오게 함과 동시에 스크린리더기에는 읽히게 하는 작업 */
    .ir {
        display: block;
        overflow: hidden;
        font-size: 0;
        line-height: 0;
        text-indent: -9999px;

    /* 이미지 스프라이트 */
    /* 아이콘 5개를 이어 만든 프레임을 SVG로 추출하여 그 좌표값을 아래와 같이

    #card__icon a {

        background-position: 0px 0;
    }
    #card__icon a.link2 {
        background-position: -80px 0;
    }
    #card__icon a.link3 {
        background-position: -160px 0;
    }
    
    /* 별점 */
    #star a {
        width: 14px;
        height: 14px;
        display: inline-block;
    }
    #star > .star_fill {
        background: url(img/star.svg);
    }
    #star > .star_empty {
        background: url(img/star_empty.svg);
    }
</style>
