HTML 기본 문서

HTML 기본 문서의 <html> 요소는 <head> 요소와 <body> 요소로 구성되어 있으며, <head> 요소 안에는 <meta>, <title>, <link>, <style>, <script> 등의 요소들이 올 수 있습니다.

요소명 설명
<meta> HTML 문서의 설명, 키워드, 문서의 작정자 등을 지정
<title> HTML 문서의 제목 지정
<link> 외부의 CSS 파일을 HTML 문서에 연결 시 사용
<style> HTML 문서 내에서 스타일(CSS)을 정의함
<script> 자바스크립트를 HTML 문서 내부에 정의, 외부에서 자바스크립트 파일을 불러올 때 사용

<html language="ko">나 <meta charset="UTF-8">처럼 태그 내부에 부가적인 설정값을 선언하는 것을 속성(attribute)이라고 하며, lang 속성은 언어를 의미합니다. 한국어는 따옴표 안에 "ko"를, 독일어의 경우 아래처럼 "de"를 넣어줍니다.

<i lang="de">Ich liebe dich</i>

<meta> 태그의 charset 속성 값 UTF-8은 '문자 인코딩(charset)'방식이라고 하며, UTF-8은 대부분의 문자를 처리 가능하기 때문에 HTML5 문자 인코딩 방식으로 주로 사용됩니다.

또, <meta> 태그에서 알아둬야 할 속성 중, name과 content 속성이 있으며 이는 나중에 학습하게 될 CSS 파트에서 반응형 웹 기술을 사용하기 위한 속성들입니다.

<meta charset="UTF-8"> <meta name="viewport"
content="width=device-width,initial-scale=1.0.minimum-scale=1.0,maximum-scale=1.0,user-scalable=no">

보통 '뷰포트'라고 하며 모바일 해상도(가로, 세로 보기)에 맞게 HTML 문서를 조절해 주는 속성들이라고 생각하면 됩니다.

HTML5 이전에 많이 사용했던 XHTML 1.0 Transitional 버전도 있는데 해당 버전의 DOCTYPE은 다음과 같으며 HTML5 문법보다 더 엄격합니다.

<DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
