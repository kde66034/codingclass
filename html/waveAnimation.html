HTML Pug 코드

HTML 코드에서는 반복 작업해야 할 것을 Pug 코드에서는 JS처럼 for문이 사용 가능하여 소스를 간략화 할 수 있습니다. circle-wrap 클래스 div 박스 하나 안에 row 클래스 div(안에 circle클래스 div 12개) 12개를 만드는 방법이다. 

div.circle-wrap
  - for (var x =1; x <= 12; x++)
    div.row
      - for (var y = 1; y <= 12; y++)
        div.circle

SCSS 코드

body에 그라데이션 배경을 주고 row 아이템들을 flex로 가운데 정렬한 다음 각각 circle에 10px의 너비값과 높이값을 주고 keyframes로 애니메이션을 주도록 합니다. 


@mixin center {
  display: flex;
  align-items: center;
  justify-content: center;
}
body {
  @include center;
  height: 100vh;
  background-image: linear-gradient(45deg, #00dbde 0%, #fc00ff 100%)
}

.row {
  display: flex;
  .circle {
    width: 10px;
    height: 10px;
    border-radius: 50%;
    margin: 5px 10px;
    background: #fff;
    transform-origin: top center;
    animation: spin 1s linear infinite;
  }
}
@keyframes spin {
    0% {transform:scale(1.1) rotate(0deg)}
    50% {transform:scale(0.2) rotate(180deg)}
    100% {transform:scale(1.1) rotate(360deg)}
  }

  // .row:nth-child(1) .circle{animation-delay: 100ms}
  // .row:nth-child(1) .circle{animation-delay: 200ms}
  // .row:nth-child(1) .circle{animation-delay: 300ms}
  // .row:nth-child(1) .circle{animation-delay: 400ms}

  // for문으로 간략화
  @for $i from 1 through 12 {
    .row:nth-child(#{$i}) .circle{animation-delay: 100ms * $i}
  }
