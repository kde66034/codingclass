함수

프로그램에서 반복적으로 사용되는 기능을 만들기 위한 코드들의 집합

01. 선언적 함수

함수를 선언할 때 이름을 붙여주는 함수. 

function func() {
    document.write("함수가 실행되었습니다.1");
}
func(); // 이걸 써야 출력됨
결과 보기
함수가 실행되었습니다.1

02. 익명 함수

말 그대로 이름 없는 함수로 함수코드가 변수명에 저장된 형태이다. 

const func = function() {
    document.write("함수가 실행되었습니다.2"); // 변수 이름 (func)을 쓰고 있어서 익명함수
}
func();
결과 보기
함수가 실행되었습니다.2

03. 매개변수 함수

함수 호출 시 인수로 전달된 값을 함수 내부에서 사용 가능하도록 해주는 변수.
str이 바로 매개변수이다. 

function func(str) {
    document.write(str);
}
func("함수가 실행되었습니다.3");
결과 보기
함수가 실행되었습니다.3

04. 리턴값 함수

함수가 기능을 수행하고 그 결과를 호출한 곳으로 돌려주는 값이다. 

function func() {
    const str = "함수가 실행되었습니다.4";
    return str;     // 실행문은 아니지만 결과값으로 저장됨
}
document.write(func());     // 도큐멘트 안에 실행문이 들어가는 형태, 리턴값이 있기에 가능함
결과 보기
함수가 실행되었습니다.4

05. 화살표 함수

기존의 함수들을 화살표 식으로 간략화 할 수 있다. 

/*
func = () => {
    document.write("함수가 실행되었습니다.5");
}
*/

// 중괄호도 생략할 수 있다.
    
func = () => document.write("함수가 실행되었습니다.5");
    
func();
결과 보기
함수가 실행되었습니다.5
