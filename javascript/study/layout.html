레이아웃 01

이번 레이아웃은 헤더, 내비게이션 바, 그리고 사이드바와 옆의 섹션, 마지막으로 아래에는 푸터 영역이 배치된 기본적이고 단순한 형태입니다. 한 번 따라해볼까요?

01. Float 방식

레이아웃 정렬 방식 중 float 방식을 이용하여 정렬해보도록 하겠습니다.

잘 정렬되다가 aside 영역과 section 영역이 나란히 붙지 못하는 모습을 볼 수 있습니다. 이 때, 두 영역을 한 줄로 나란히 배치하기 위해서는 해당 aside, section 영역 속성에 float: left;를 설정해줍니다.
하지만 그렇게 되면 section 밑의 footer 영역의 height 값이 0이 되어버리는 버그가 발생하는데 그럴 때에는 버그가 걸린 해당 영역 footer에 clear: both; 를 설정해주시면 됩니다.

자바스크립트 기본 규칙

1. 세미콜론(;)

하나의 실행문이 끝나면 세미콜론(;)을 붙여줍니다. 붙이지 않아도 무관하나 처음 학습한다면 붙이는 것을 권장합니다.

var num = 5;
document.write(num);

2. 대소문자 구분

자바스크립트 명령어는 꼭 대소문자를 구분합니다.

var num = 5;
document.write(num) (O)
document.write(Num) (X)

3. 들여쓰기

가독성을 위해 들여쓰기를 합니다.

var avgDistance = function() {
      var speed = distance * time;
}

4. 주석

일반적으로 특정 코드에 대한 설명을 입력할 때나 특정 부분 코드가 실행되지 않도록 할 때 사용하며, 이 주석 표시는 프로그램에 영향을 끼치지 않습니다.

주석 처리 방법은 주석 처리할 줄을 클릭 후 Ctrl + [/]키를 누르시면 됩니다.

// 한 줄 주석 처리 방법

/* 여러 줄
    주석 처리 방법 */

출력하는 방법

자바스크립트에서는 출력 방법이 여러가지 있으며 이는 기본 명령어입니다.

분류 메소드
입력 prompt()
출력 document.write()
alert()
console.log()
확인 confirm() 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>출력 방법 : 기본 명령어</title>
    <script>
        prompt('입력 제목', '입력 내용');
        document.write('출력 내용')
        alert('출력 내용');
        console.log('출력 내용');
        confirm('확인 내용');
    </script>
</head>
<body>
</body>
</html>

자바스크립트 사용 방법

자바스크립트의 사용법에는 ①외부 파일로 로드하는 방법, ②script 태그에 사용하는 방법, ③body 태그에 직접 기술하는 방법 등이 있습니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>Javasript</title>
    <!-- 1. 외부 파일로 로드하는 방법 -->
    <script type="text/Javasript" src="js/script.js"></script>
    <!-- 2. script 태그 사이에 기술하는 방법 -->
    <script>
        document.write("Javasript");
    </script>
</head>
<body>
    <!-- 3. body 태그에 직접 기술하는 방법 -->
    <a href="#" onclick="alert('Javasript')"></a>
</body>
</html>
