데이터 타입

변수에 저장되는 데이터의 유형(타입)을 의미하며 원시(Primitive) 타입, 객체(Object)타입으로 나뉩니다.

원시 타입에는 number, string, boolean, undefined, null, symbol 등이 있으며 객체 타입에는 function, object, array 등이 있습니다.

01. number

정수, 소수점, 지수 표현 가능합니다.

var num1 = 10;
var num2 = 10.5;
var num3 = le+2;

console.log(num1); // 10
console.log(num2); // 10.5
console.log(num3); // 100 le+2는 1*10의 2승을 의미함.

02. string

문자를 표현하고자 할 때 사용하는 데이터이며
' '(작은따옴표) 혹은 " "(큰따옴표)로 표현합니다.

var str1 = ' 문자열 ' ;
var str2 = " 문자열 " ;

문자열 안에 문자열을 넣는 방법
var str1 = '문자 "문자"';
var str1 = "문자 "문자'";

03. boolean

참(true)과 거짓(false)을 나타내는 데 사용

var temp1 = (5 > 4);
var temp2 = (5 < 4);

console.log(temp1); // 결과 : 참
console.log(temp2); // 결과 : 거짓

04. undefined

변수 선언하고 데이터 값이 미지정 상태일 때 결과 값으로 나타나는 데이터입니다.

var temp1 = 10;
var temp2;

console.log(temp1); // 결과 : 10
console.log(temp2); // 결과 : undefined

05. null

데이터에 값을 저장했지만 값이 존재하지 않을 때, 혹은 변수를 빈 상태로 초기화 했을 때 나타나는 데이터입니다.

var obj = 10;
obj = null;
console.log(obj); // 결과 : null | 변수를 빈 상태로 만듬

var obj = document.getElementById('gnb');
console.log(obj); // 결과 : 변수에 데이터 값을 저장했지만 값이 존재하지 않아 null로 저장
