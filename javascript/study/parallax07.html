패럴럭스 이펙트 07

이번 시간에는 그 동안 배웠던 JavaScript를 이용하여 패럴럭스(parallax, 패럴랙스) 이펙트를 만들어보도록 하겠습니다. 먼저 parallax란, '시차(視差)'라는 의미를 가진 천문학 용어이며, 이번 포스팅에서 내비게이션 바에서 특정 섹션으로 가는 버튼을 누르면 해당 섹션으로 시차를 두고 스무스하게 이동시켜 주는 효과를 만들어보고자 합니다.

이번 패럴럭스 이펙트 07에서는 리빌(reveal) 효과 애니메이션을 주겠습니다.

HTML 소스

애니메이션을 주고자 하는 이미지와 텍스트 영역에 reveal 클래스값을 준다.

· reveal-RTL : 우측에서 좌측으로 리빌 효과
· reveal-TTB : 상단에서 하단으로 리빌 효과
· reveal-BTT : 하단에서 상단으로 리빌 효과
· reveal-TWO : 두 가지 애니메이션을 동시에 줌

JS 소스

reveal 클래스 자식에 글씨가 있는 경우 그 글씨를 span 태그로 감싸도록 스크립트를 짜고, 스크롤 값이 섹션 영역에 도달했을 경우 show 클래스값을 붙여 리빌 효과를 주도록 스크립트를 짰다. 

<script>
    const revealText = document.querySelectorAll(".reveal");

    // reveal 클래스 자식에 글씨가 있는 경우, 가상으로 span 태그를 만들어서 넣기
    revealText.forEach((el)=>{
        let splitText = el.innerText;           

        // console.log(splitText)
        if(splitText){
            el.innerHTML = '<span>' + splitText + '</span>';
        }
    });  
    
    function scroll() {
        let scrollTop = window.scrollY;     // 스크롤탑값
        
        const reveal = document.querySelectorAll(".reveal");    // 리빌 효과 상수

        reveal.forEach(el => {
            let revealOffset = el.offsetTop + el.parentElement.offsetTop;
            let revealDelay = el.dataset.delay;     // 딜레이 주기

            // if(scrollTop > revealOffset - window.innerHeight/2){
            //     el.classList.add("show");     // 스크롤 위치가 영역에 도달할 시 show 붙여주기
            // }

            if(scrollTop >= revealOffset - window.innerHeight){
                if(revealDelay == undefined){
                    el.classList.add("show");
                } else {
                    setTimeout(() => {
                        el.classList.add("show");
                    }, revealDelay)
                }
            }
        });

        requestAnimationFrame(scroll);
    }
    scroll();
</script>
