jQuery 필터 선택자

제이쿼리에서 사용되는 필터 선택자에 대해 살펴봅시다.
선택자에 ' : '가 붙습니다.

필터 선택자

선택자 종류 선택자 설명
:even $("tr:even") tr 요소 중 짝수 행만 선택
:odd $("tr:odd") tr 요소 중 홀수 행만 선택
:first $("tr:first") 첫 번째 td 요소를 선택
:last $(":header") 헤딩(h1~h6) 요소를 선택
:eq() $("li:eq(0)") index가 0인 li 요소를 선택. index는 0번이 첫 번째 요소입니다.
:gt() $("li:gt(0)") index가 0보다 큰 li 요소들을 선택
:lt() $("li:lt(2)") index가 2보다 작은 li 요소들을 선택
:not() $("li:not(.bg)") li 요소 중에서 class명 bg가 아닌 li 요소를 선택
:root $(":root") html을 의미함
:animated $(":animated") 움직이는 요소를 선택

예제 코드

<!DOCTYPE html> 
<html lang="ko"> 
<head> 
    <meta charset="UTF-8"> 
    <title>필터 선택자<title>

        });
    </script> 
</head> 
<body>

자식 필터 선택자

: 'child'가 붙은 선택자는 요소가 순차적으로 나열되어 있을 때 사용하며, "of-type"이 붙은 선택자는 요소가 순차적으로 나열되어 있지 않아도 동일 요소라면 선택 가능하다.

선택자 종류 선택자 설명
:first-child $("span:first-child") 첫 번째 span 요소를 선택
:last-child $("span:last-child") 마지막 span 요소를 선택
:first-of-type $("span:first-of-type") span 요소 중에서 첫 번째 span 요소를 선택
:last-of-type $("span:last-of-type") span 요소 중에서 마지막 span 요소를 선택
:nth-child() $("span:nth-child(2)") 두 번째 span 요소를 선택함.
nth-child(2n)은 2, 4, 6, …번째 요소를 선택하고,
nth-child(2n+1)은 1, 3, 5, …번째 요소를 선택함.
:nth-last-child() $("span:nth-last-child(2)") 마지막에서 두 번째 span 요소를 선택
:nth-of-type() $("span:nth-of-type(2)") span 요소 중에서 두 번째 span 요소를 선택
:nth-last-of-type() $("span:nth-last-of-type(2)") span 요소 중에서 마지막에서 두 번째 span 요소를 선택
:only-child $("div > span:only-child") div의 자식 요소에서 오직 span 요소가 하나만 있는 span 요소를 선택
:only-of-type $("div > span:only-of-type") div의 자식 요소에서 span 요소가 하나만 있는 span 요소를 선택

예제 코드

콘텐츠 필터 선택자

선택자 종류 선택자 설명
:contains() $("p:contains('html')") p 요소 중에서 'html' 텍스트를 포함하고 있는 p 요소를 선택
:empty $("div:empty") div 요소 중에서 자식 요소가 없는 div 요소를 선택
:parent $("span:parent") span 요소 중에서 자식 요소가 있는 span 요소를 선택
:has() $("section:has(article)") section 요소 중에서 article을 하위 요소로 가지고 있는 section 요소를 선택

예제 코드

<!DOCTYPE html> 
<html lang="ko"> 
<head> 
    <meta charset="UTF-8"> 
    <title>자식 필터 선택자<title> 
    <style> 
        * {
            margin: 5px;
        }
        #m1 > p:last-child{
            height: 20px;
        }
        span {
            display: inline-block;
            height: 20px;
        }
    </style> 
    <script src="jquery-3.3.1.min.js"></script> 
    <script> 
        $(document).ready(function() {
            $("#m1 > p:contains('html')").css("border","4px solid red");
            $("#m1 > p:empty").css("border","4px solid orange");
            $("#m2 > span:parent").css("border","4px solid yellow");
            $("#m3 > section:has(article)").css("border","4px solid green");
        });
    </script> 
</head> 
<body> 
    <div id="m1">
        <p>html, css, javascript</p>
        <p>html5와 웹표준</p>
        <p></p>
    </div>
    <div id="m2">
        <span></span>
        <span>내용1</span>
    </div>
    <div id="m3">
        <section>
            <article>내용2_1</article>
        </section>
        <section>
            <div>
                <article>내용2_2</article>
            </div>
        </section>
    </div>
</body> 
</html>

폼 필터 선택자

선택자 종류 선택자 설명
:text $("input:text") <input type="text"> 요소를 선택
:password $("input:password") <input type="password"> 요소를 선택
:image $("input:image") <input type="image"> 요소를 선택
:file $("input:file") input type="file"> 요소를 선택
:button $(":button") <input type="button">, <button> 요소를 선택
:checkbox $("input:checkbox") <input type="techeckboxxt"> 요소를 선택
:radio $("input:radio") <input type="radio"> 요소를 선택
:submit $("input:submit") <input type="submit"> 요소를 선택
:reset $("input:reset") <input type="reset"> 요소를 선택
:input $(":input") 모든 <input> 요소를 선택
:checked $("input:checked") <input> 요소에 checked 속성이 있는 요소를 선택
:selected $("option:selected") <option> 요소에 selected 속성이 있는 요소를 선택
:focus $("input:focus") 현재 <input>에 포커스가 있는 요소를 선택
:disabled $("input:disabled") <input> 요소에 disabled 속성이 있는 요소를 선택
:enabled $("input:enabled") <input> 요소 중 disabled가 아닌 요소를 선택

예제 코드

<!DOCTYPE html> 
<html lang="ko"> 
<head> 
    <meta charset="UTF-8"> 
    <title>폼 필터 선택자<title> 
    <script src="jquery-3.3.1.min.js"></script> 
    <script> 
        $(document).ready(function() {
            $("input:text").css("background","red");
            $("input:password").css("background","orange");
            $(":button").css("background","yellow");
            $("input:checked + label").css("background","green");
            $("option:selected").css("color","blue");
            $("textarea:disabled").css("background","pink");
        });
    </script> 
</head> 
<body> 
    <div><input type="text"></div>
    <div><input type="password"></div>
    <div><button>확인</div>
    <div><input type="checkbox" id="css" checked="checked"><label for="css">css</label></div>
    <div>
        <select>
            <option>과목선택</option>
            <option>javascript</option>
            <option selected="selected">jQuery</option>
        </select>
        <textarea cols="20" id="" rows="5" disabled="disabled">javascript</textarea>
    </div>
</body> 
</html>

가시성 필터 선택자

선택자 종류 선택자 설명
:hidden div:hidden div 요소 중 hidden인 요소를 선택
:visible div:visible div 요소 중 visible 요소를 선택

예제 코드

<!DOCTYPE html> 
<html lang="ko"> 
<head> 
    <meta charset="UTF-8"> 
    <title>가시성 필터 선택자<title> 
    <style> 
        #vis {
            display: none;
        }
    </style>
    <script src="jquery-3.3.1.min.js"></script> 
    <script> 
        $(document).ready(function() {
            console.log($("div:hidden").text());
            $("div:visible").css("background","orange");
        });
    </script> 
</head> 
<body> 
    <div id="vis">내용1</div>
    <div>내용2</div>
</body> 
</html>
