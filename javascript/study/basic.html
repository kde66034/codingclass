기본 규칙

자바스크립트를 다루기 위해서는 먼저 기초적 특징과 기본 문법을 숙지할 필요가 있습니다.

자바스크립트란?

JavaScript는 웹 페이지에서 복잡한 기능을 구현할 수 있도록 하는 스크립트 프로그래밍 언어입니다. HTML은 정적인 레이아웃을, 자바스크립트는 동적인 기능을 표현합니다.

자바스크립트는 1995년 넷스케이프 커뮤니케이션의 브렌던 아이크가 개발하였스며 Netscape Navigsator2.0에 구현되었습니다. 1996년에는 마이크로소프트사의 Internet Explorer3.0 탑재되었으며 java 애플릿으로 구현된 웹 브라우저의 지지부진한 보급과 맞물려 빠르게 보급 되었습니다. 1997년부터 ESMAScript를 따르는 표준화 작업을 통해 현재는 모든 브라우저에서 지원됩니다.

자바스크립트 기본 규칙

1. 세미콜론(;)

하나의 실행문이 끝나면 세미콜론(;)을 붙여줍니다. 붙이지 않아도 무관하나 처음 학습한다면 붙이는 것을 권장합니다.

var num = 5;
document.write(num);

2. 대소문자 구분

자바스크립트 명령어는 꼭 대소문자를 구분합니다.

var num = 5;
document.write(num) (O)
document.write(Num) (X)


3. 들여쓰기

가독성을 위해 들여쓰기를 합니다.

var avgDistance = function() {
      var speed = distance * time;
}


4. 주석

일반적으로 특정 코드에 대한 설명을 입력할 때나 특정 부분 코드가 실행되지 않도록 할 때 사용하며, 이 주석 표시는 프로그램에 영향을 끼치지 않습니다.

주석 처리 방법은 주석 처리할 줄을 클릭 후 Ctrl + [/]키를 누르시면 됩니다.

// 한 줄 주석 처리 방법

/* 여러 줄
    주석 처리 방법 */

출력하는 방법

자바스크립트에서는 출력 방법이 여러가지 있으며 이는 기본 명령어입니다.

분류 메소드
입력 prompt()
출력 document.write()
alert()
console.log()
확인 confirm() 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>출력 방법 : 기본 명령어</title>
    <script>
        prompt('입력 제목', '입력 내용');
        document.write('출력 내용')
        alert('출력 내용');
        console.log('출력 내용');
        confirm('확인 내용');
    </script>
</head>
<body>
</body>
</html>

자바스크립트 사용 방법

자바스크립트의 사용법에는 ①외부 파일로 로드하는 방법, ②script 태그에 기술하는 방법, ③body 태그에 직접 기술하는 방법 등이 있습니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>Javasript</title>
    <!-- ①외부 파일로 로드하는 방법 -->
    <script type="text/Javasript" src="js/script.js"></script>
    <!-- ②script 태그에 기술하는 방법 -->
    <script>
        document.write("Javasript");
    </script>
</head>
<body>
    <!-- ③body 태그에 직접 기술하는 방법 -->
    <a href="#" onclick="alert('Javasript')"></a>
</body>
</html>
