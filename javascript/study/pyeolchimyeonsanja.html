펼침연산자 - 복사, 추가, 결합

이번 포스팅에서는 펼침연산자에 대해 다뤄보도록 하겠습니다.

펼침연산자란 간단히 말하자면 배열에 포함된 항목을 목록으로 바꿔주는 연산자입니다. 마침표 세 개(…)로 표시하며, 단독으로 쓰일 수 없습니다. 그리고 변수/상수 자체가 아닌 객체나 배열 안에 담아줘야만 합니다.

const class1 = [1, 2, 3];

const a = ...class2; ❌ (틀린 방법!)
const a = [...class2]; ⭕ (올바른 사용법)

펼침연산자 - 복사

변수 또는 상수 안의 데이터를 그대로 가져와서 사용하는 방식이다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    const spread = { ...obj }

    document.write(spread.a);
    document.write(spread.b);
    document.write(spread.c);
}
결과보기
100
200
javascript

펼침연산자 - 추가

변수 또는 상수 안의 데이터를 그대로 가져온 다음 추가로 데이터를 생성하는 방법이다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    const spread = { ...obj, d: "jquery" }

    document.write(spread.d);
}
결과보기
jquery

펼침연산자 - 결합

변수 또는 상수가 여러개일 경우 이를 하나로 결합하여 사용할 수 있다. 

{
    const objA = {
        a: 100,
        b: 200
    }
    const objB = {
        c: "javascript",
        d: "jquery"
    }
    const spread = { ...objA, ...objB }

    document.write(spread.a);
    document.write(spread.b);
    document.write(spread.c);
    document.write(spread.d);
}
결과보기
100
200
