퀴즈 만들기 03

이번 시간에는 그 동안 배웠던 JavaScript를 이용하여 퀴즈를 만들어보도록 하겠습니다.
이번 학습 내용을 잘 활용하여 웹디자인기능사 시험 공부 열심히 해 봅시다!
변수(let)와 상수(const)에 쿼리 셀렉터 값을 저장시키고, 그 동안 배운 스크립트를 통해 문제 정보를 HTML 태그에 출력시키고, 정답을 숨겼다가 버튼을 눌러 보이게 하는 등의 다양한 효과들을 구현시켜봅시다!

주관식(여러 문제) 유형

세 번째 주관식(여러 문제) 유형의 자바스크립트 소스와 부연 설명(주석 처리됨)이다.
HTML과 CSS 소스를 함께 보고 싶다면 맨 아래의 '원본 소스 보기' 버튼을 누를 것. 

// 선택자 : 상수 const를 필요한 만큼 선언한 다음 HTML 영역의 div, 시맨틱 태그 등의 Class명과 연결시킨다.
// 문제(<div class="quiz">)가 여러개이므로 뒤에 All을 꼭! 붙여주기
const quizType = document.querySelectorAll(".quiz__type");                 // 퀴즈 종류
const quizNumber = document.querySelectorAll(".quiz__question .number");   // 퀴즈 번호
const quizAsk = document.querySelectorAll(".quiz__question .ask");         // 퀴즈 질문
const quizConfirm = document.querySelectorAll(".quiz__answer .confirm");   // 정답 확인 버튼
const quizResult = document.querySelectorAll(".quiz__answer .result");     // 정답 결과
const quizInput = document.querySelectorAll(".quiz__answer .input");       // 사용자 정답
const quizView = document.querySelectorAll(".quiz__view");             // 강아지

// 문제 정보

문제를 출력했다면 일단 정답 확인 버튼을 누르기 전까지 보이게 해서는 안된다.
정답 숨기는 방법도 원시적 방법과 for문, forEach 이렇게 3가지가 있으며 최종적으로는 forEach를 사용한다. 

// 정답 확인 : forEach
// 요소 btn을 만들어서, 선택자 quizConfirm 과 연결시킴
// 펑션 괄호 안의 순서는 반드시 (element, index, array)이기때문에 자동으로 btn=element, num=index가 됨.
quizConfirm.forEach((btn, num) => {
    btn.addEventListener("click", () => {       // 클릭 시 이벤트 발생 시키기
            
        // 사용자 정답
        const userWord = quizInput[num].value; // 사용자가 입력한 답을 userWord라는 상수에 저장!

        // 사용자 정답 == 문제 정답 (맞는지 확인)
        if(userWord == quizInfo[num].answerResult) {
            //정답이면
            // alert("정답");   // alert 테스트용, 테스트 마친 후 주석처리
            quizView[num].classList.add("like");    // 강아지 미소
            quizConfirm[num].style.display = "none";    // 정답확인버튼 숨김
            quizResult[num].style.display = "none";     // 정답 숨기기
            quizInput[num].style.display = "block";     // 입력창 남기기
        } else {
            //오답이면
            // alert("오답");
            quizView[num].classList.add("dislike");     // 강아지 슬픔
            quizConfirm[num].style.display = "none";    // 정답확인버튼 숨김
            quizResult[num].style.display = "block";    // 정답 표시
            quizInput[num].style.display = "none";      // 입력창 숨김
        }
    });
});

이제 정답 입력 후의 동작을 만들어야 한다. 식이 복잡해지기 때문에 반드시 forEach문을 사용해야 할 것.
addEventListener를 통해 클릭 시의 이벤트를, if문을 통해 사용자의 입력 데이터와 문제 정답을 비교하고 그 다음 나와야 할 동작들을 위의 예시 코드와 같이 구현하도록 한다.
